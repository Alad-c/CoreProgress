# 💪 CoreProgress | Fitness Tracker

CoreProgress is a full-stack web application designed to track physical activities, specifically focusing on Street Workout, Squash, and Running. 

## ✨ Features
* **Street Workout:** Track the number of sets and use a built-in rest timer (3 minutes).
* **Squash Match:** Keep score of current matches and track overall win/loss statistics.
* **Running:** Log 5km runs and track progress towards a 20km weekly goal.
* **User Authentication:** Secure registration and login system with password hashing.
* **Leaderboard:** View the Top 5 athletes based on their saved workout logs.

## 🛠️ Technologies Used
* **Frontend:** HTML5, CSS3, Vanilla JavaScript
* **Backend:** Node.js, Express.js
* **Database:** SQLite3
* **Security:** bcrypt (for password hashing)
